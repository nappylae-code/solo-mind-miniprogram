# Nearmi 🌿
> 你不是一个人。

无论你身边有没有人，内心的孤独感都真实存在。Nearmi 陪你记录每一天的心情，关注你真实的内心世界。

---

## 📱 功能

### ✅ 已上线

**登录** — 设置头像和昵称，一次设置永久使用

**今天** — 每日情绪打卡（😄 棒极了 / 🙂 开心 / 😌 平静 / 😢 难过 / 😠 生气），支持简短备注

**周历视图** — 查看本周每天的情绪记录

**连续打卡** — 追踪你坚持打卡的天数 🔥

**每日语录** — 每天一句，陪你面对独处时光

**日记列表** — 按月分组展示所有日记，支持情绪筛选和关键词搜索

**日记详情** — 查看单篇日记内容，支持编辑和删除

**日记编辑** — 写日记并关联当日情绪，保存至云端

**我的** — 个人主页，查看打卡天数、日记篇数、连续记录，以及成就勋章

**情绪报告** — 按本周 / 本月 / 本年查看情绪分布

**广场** — 匿名分享今日心情，用预设表情回应其他用户
